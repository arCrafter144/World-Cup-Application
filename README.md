# ⚽ World Cup Application

A modern football application that provides all World Cup information in one place.

## ✨ Features

* 📅 Match schedules
* 📊 Win probability predictions
* 👥 Team lineups and player information
* ⭐ Player of the Match voting
* 📈 Live match statistics
* 🏆 Group standings and tournament tables
* 🔍 Search for teams, players, and matches
* 🎨 Dream Team Builder

## 🎯 Goal

This project was created to help football fans follow the World Cup without constantly searching the web for schedules, statistics, lineups, and predictions.

## 🛠️ Tech Stack

* Frontend: Angular
* Backend: Spring Boot
* Database: PostgreSQL
* Authentication: JWT

## 🚀 Future Improvements

* Match notifications
* Multi-language support
* Advanced player analytics
* User accounts and profiles
* AI-powered match predictions
